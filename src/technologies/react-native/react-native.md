### Documentation officielle

- [React Native Docs](https://reactnative.dev/docs/getting-started) — Référence composants et APIs
- [Expo Docs](https://docs.expo.dev/) — Tout sur Expo, très bien écrit
- [Expo Router](https://docs.expo.dev/router/introduction/) — Navigation file-based
- [TanStack Query](https://tanstack.com/query/latest/docs/react/overview) — Data fetching

### Cours recommandés

- [React Native - The Practical Guide](https://www.udemy.com/course/react-native-the-practical-guide/) (Academind) — Complet, mise à jour régulière
- [Expo in 100 Seconds](https://www.youtube.com/watch?v=vFW_TxKLyrE) — Intro rapide
- [Notjust.dev YouTube](https://www.youtube.com/@notjustdev) — Projets concrets, moderne

- [expo/examples](https://github.com/expo/examples) — Templates officiels
- [react-native-community/cli](https://github.com/react-native-community/cli) — Si tu veux comprendre le CLI vanilla
- [wcandillon/can-it-be-done-in-react-native](https://github.com/wcandillon/can-it-be-done-in-react-native) — Animations avancées
